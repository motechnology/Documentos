Termo de Encerramento

1. Objetivos

Este documento visa detalhar o encerramento do projeto de Rede de sensores remotos com Arduinos desenvolvido para a disciplina de Projeto Integrador II do curso de Engenharia de Telecomunicações - IFSC/SJ. Será relatado todo o processo de construção dos módulos pertencentes ao sistemas assim como as dificuldades e aprendizados adquiridos neste período.

2. Planejado vs Realizado

O funcionamento principal estabelecido no início do projeto foi atingido conforme esperado, porém com modificações em alguns equipamentos para uma melhor eficiência e facilidade para o usuário entender a construção do mesmo. Uma das modificações foi a troca do módulo ESP8266 e o Arduino Mega presentes na estação receptora pelo componente ESP8266 nodemcu, capaz de substituí-los e ainda ter um consumo energético menor.

2.1 Os objetivos foram concluídos?

Sim, todos os módulos estão funcionando conforme esperado.

2.2. O projeto foi entregue dentro do prazo?

Sim, o projeto foi entregue em funcionamento dentro da data prevista no dia 05/12/19.

3. O projeto

O projeto trouxe grandes experiências pessoais e técnicas para todos os participantes da equipe. Com o auxílio do P.O. foi possível otimizar o trabalho para uma melhor rentabilidade de trabalho e realizar todas as entregas nos prazos adequados. A seguir, segue alguns pontos referentes ao desenvolvimento de todo o projeto.

3.1 Pontos fortes

- Característica individuais resultaram em grandes melhorias para o projeto como um todo;
- A organização da equipa para a conclusão foi de extrema importância para o sucesso obtido;
- Novos conhecimentos sobre métodos ágeis;
- O projeto foi dividido em pequenas partes que se fundiram para formar módulos maiores facilitando o trabalho de todos.

3.2 Pontos fracos

- Choque de horários por parte dos integrantes da equipe prejudicou em algumas reuniões semanais para debate do andamento do projeto;
- Períodos sem trabalhar prejudicaram em partes o andamento do projeto.
